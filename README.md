# React Native Product Module

## Overview
A simple React Native application demonstrating a product listing and detail flow using a public API.

## Features
- Product list with loading, error, and empty states
- Product detail screen
- Stack navigation
- Pull-to-refresh
- Clean architecture with Zustand state management

## Tech Stack
- React Native (Expo)
- TypeScript
- Zustand
- Axios
- React Navigation
